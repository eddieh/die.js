embec.js

Embedded EcmaScript - a JavaScript templating engine

Intro

Embec is dumb and probably isn't what you're looking for. It's a template engine with no frills, no gulps, webpacks, Jakes, YAML, package.jsonâ€”it's just one small file and no dependencies.

What can it do? All the normal templating stuff: interpolate, flow control, includes, and blocks.

What it will never do? File IO, impose a stucture or a style on your project.

Don't use Embec! It isn't what you're looking for. Don't you want to use React/JSX? That way you can write <GarbageComponent attr="yuck"> like is '96 and SGML is all the rage?

Tags

Embec tags are where you tell the template compiler to stop copying and to start doing some processing. All tags start with <% and end with %>.

Description

Embec has several tags.

<% %>
Statement tag
<%= %>
Interpolate tag
<%# %>
Comment tag
